# SQL Music Store Analysis

## Project Overview

This project analyzes a Music Store database using PostgreSQL to answer business-focused questions related to customer behavior, sales performance, revenue generation, and music preferences.

The project was completed as part of my SQL learning journey by following and implementing concepts from an online tutorial. I executed the queries, explored the database, and analyzed the results to strengthen my understanding of SQL and data analysis.

## Tools Used

- PostgreSQL
- SQL
- pgAdmin
- GitHub

## SQL Concepts Applied

- Joins
- Common Table Expressions (CTEs)
- Subqueries
- Aggregate Functions
- Window Functions
- Group By & Having
- Data Filtering & Sorting

## Business Questions Solved

- Identify the senior-most employee.
- Find countries with the highest number of invoices.
- Determine top invoice values.
- Identify the highest revenue-generating city.
- Find the best customer based on total spending.
- Analyze Rock music listeners.
- Identify top-performing Rock artists.
- Compare track lengths with average song duration.
- Analyze customer spending by artist.
- Find the most popular music genre by country.
- Identify top-spending customers in each country.

- Identified high-value customers and spending patterns.
